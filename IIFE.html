Immediately Invoked Function Expression

Benefits:

  • Makes the code easily maintenable
  • Makes the code organizable
  • Does not create unnecessary global variables and functions
  • Functions and variables defined in IIFE do not conflict with other functions and variables. Even if they have the same name

Syntax:

(function () { // code goes here… })();

Or

arrow functions like: (() => { // code goes here… })();
